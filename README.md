# Dice Roller

A simple browser-based dice roller built with HTML, CSS, and JavaScript.  
The user can choose how many dice to roll, click a button, and see both the numeric results and matching dice images.

## Features

- Choose how many dice to roll
- Random dice values from 1 to 6
- Displays the rolled numbers as text
- Shows matching dice face images
- Simple and lightweight interface

## Built With

- HTML
- CSS
- JavaScript

## Project Structure

- dice.html
- dice.css
- dice.js
- dice_images/

## How to Run

1. Download or clone the project folder.
2. Open `dice.html` in your browser.
3. Enter the number of dice you want to roll.
4. Click the Roll button.

## How It Works

- The user enters a number in the input field.
- JavaScript generates random values between 1 and 6.
- The results are shown on the page as text.
- The corresponding dice images are loaded from the `dice_images` folder.
